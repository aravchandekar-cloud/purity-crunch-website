## Goal
Establish a new professional visual identity for Purity Crunch — new color palette, new font stack, and a new SVG logo — replacing the current warm-editorial design with a cleaner, more corporate B2B-credible look.

## Purpose
B2B buyers judge supplier credibility in seconds. The current warm/artisan aesthetic needs to shift toward a polished, modern, trustworthy visual identity that signals export-readiness and professionalism.

## Output
- Updated CSS custom properties (colors, fonts)
- New Google Fonts link
- New inline SVG logo placed in navbar and footer
- All existing sections inherit new identity via CSS variables
</objective>

  <files>css/styles.css</files>
  <action>
    Replace all CSS custom properties in :root with a new professional B2B palette:
    - Choose a sophisticated, trust-signaling palette (deep navy/slate primary, clean whites, accent that conveys premium quality)
    - Update --primary, --primary-hover, --primary-accessible, --secondary, --accent, --accent-light, --accent-green
    - Update --bg-color, --surface, --surface-dark, --text-main, --text-muted, --border
    - Update all shadow variables to use new brand colors
    - Search for any hardcoded color values in styles.css AND index.html (inline styles, style attributes) and replace with new palette or CSS variables <!-- audit-added: index.html may have inline color values -->
    - Document the old color token values as a comment block at the bottom of styles.css for rollback reference <!-- audit-added -->
    - Ensure WCAG AA contrast ratios (4.5:1 for text, 3:1 for large text/UI)
    Avoid: Changing layout, spacing, or animation properties — only colors
  </action>
  <verify>Open index.html in browser — all sections show new color scheme with no old orange/espresso/gold remnants. Check contrast with browser DevTools accessibility audit.</verify>

    - Update font-weight values if new fonts have different weight ranges
    - Update the CSS comment at top of styles.css to reflect new font names
    Avoid: Changing font-size, line-height, letter-spacing, or any spacing/layout properties
